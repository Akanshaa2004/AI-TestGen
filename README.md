## AI-TestGen

AI‑powered tool to automatically generate unit tests for your JavaScript/React code.

## Features

- Parses React components and extracts testable logic
- Generates Jest / Vitest test suites
- Supports hooks, async functions, and event handlers
- Simple UI to review and edit generated tests

## Tech stack

- React (Vite)
- TypeScript (or JavaScript – adjust)
- Some AI API (OpenAI / Gemini / local model)

## Getting started

```bash
# Clone the repo
git clone https://github.com/Akanshaa2004/AI-TestGen
cd AI-TestGen

# Install dependencies
npm install

# Run dev server
npm run dev
